# KwaValidator - Node.js Implementation

This folder contains the **Node.js/TypeScript** implementation of the KwaValidator library.

## Purpose
A professional-grade library for validating Angolan documents (ID/BI, NIF, Passport) within the Node.js ecosystem. 

## Features
- **Offline Validation:** Fast, local format checking using optimized regex.
- **Online Validation:** API-based verification for document existence.
- **TypeScript Support:** Fully typed for a better developer experience.

## Ecosystem
This is part of the [KwaValidator Multi-platform project](https://github.com).
