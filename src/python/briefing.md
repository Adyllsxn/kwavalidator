# KwaValidator - Python Implementation

This folder contains the **Python** implementation of the KwaValidator library.

## Purpose
A specialized package for validating Angolan documents (ID/BI, NIF, Passport) for Python applications, from scripts to web frameworks like FastAPI or Django.

## Features
- **Native Performance:** Pythonic implementation with zero heavy dependencies.
- **Dual Mode:** Choose between local format validation (Offline) or real-time API verification (Online).
- **Standards Compliant:** Follows Angolan document formatting rules (AGT and Ministry of Justice).

## Ecosystem
This is part of the [KwaValidator Multi-platform project](https://github.com).
